# Chicago Crime Clustering Project
## Overview
Distributed clustering of 7.9M Chicago crime incidents using PySpark MLlib.
## Structure
- notebooks/ - Jupyter notebooks for ingestion, features, training, evaluation
- tableau/ - Tableau workbooks and documentation
- scripts/ - Pipeline scripts and profiler
- config/ - Spark and Tableau configuration
- data/ - Schemas and sample data
- tests/ - Pipeline tests
## How to Run
1. Open Google Colab and run the single-cell notebook
2. Download output CSVs for Tableau
3. Open Tableau Public and connect to CSVs
## Requirements
- PySpark 3.5+
- scikit-learn 1.3+
- Google Colab (free tier)
- Tableau Public 2024+
